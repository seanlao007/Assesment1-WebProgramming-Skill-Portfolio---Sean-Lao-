Section 1: HTML Tag Examples

1. The Document Structure (html, head, title, link, body)

These tags provide the essential "skeleton" of the website. The head section handles background information like the tab title and the link to your external CSS file, while the body section holds all the actual content that the user sees on the screen.

4. Media & Links (a, iframe)

The anchor tag (a) allows you to create hyperlinks to move between different pages of your site. The iframe tag is used specifically to embed external media, such as the MP4 videos used to demonstrate the baking process.

5. Formatting Code (pre, code)

These tags are specifically designed to display technical information. The pre tag preserves the exact spacing and line breaks of your text, while the code tag applies a monospaced font to make the text look like it was written in a programming editor.

Section 2: CSS Property Examples

1. Layout (display: grid, grid-template-columns, gap)

These properties are the foundation of a modern layout. They turn a container into a grid system, allowing you to define exactly how many columns you want (using fr units) and how much breathing room (gap) should exist between your cards.


.recipe-grid {
    display: grid;
    grid-template-columns: 1.2fr 1.2fr 0.8fr;
    gap: 30px;
}

2. Positioning (position: sticky, z-index)

These properties control how elements interact when the user scrolls. Sticky allows the header to remain visible at the top of the viewport, while z-index acts as a "layering" tool to ensure the header stays in front of the content sliding underneath it.


header {
    position: sticky;
    top: 0;
    z-index: 1000;
}

3. Visuals (background-image, border-radius, text-transform)

These properties handle the aesthetic details of your site. They allow you to add decorative images to sections, round off the sharp corners of your recipe cards for a softer look, and automatically capitalize titles for a consistent design.


.recipe-card {
    background-image: url('cookies.jpg');
    border-radius: 15px;
}
h2 {
    text-transform: uppercase;
}

4. Sizing (box-sizing, margin, padding)

Spacing properties are used to prevent content from looking cluttered. Margin creates space outside of elements, while padding creates space inside. Using border-box is a "best practice" that ensures your padding doesn't accidentally make your elements wider than your grid columns.


* {
    box-sizing: border-box;
}
section {
    padding: 20px;
    margin-bottom: 30px;
}

5. Typography (font-family, text-shadow)

Typography properties set the professional tone of the website. Font-family defines the typeface used for the text, and text-shadow is a creative way to add a "glow" or shadow behind headers to make them pop against busy background images.


body {
    font-family: Arial, sans-serif;
}
h1 {
    text-shadow: 2px 2px 4px #000000;
}
